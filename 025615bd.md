## 1. What This Is

`webba` is a two-file Python library for intelligent web search and content fetching.

**Key properties:**
- Works with **zero API keys** — DDG + SearXNG (auto-started via Docker) + Google scrape give full quality results out of the box
- Adds paid providers (Serper, Tavily, Exa, Perplexity, Brave) when keys are present — quota tracked in SQLite
- Fetches any URL (GitHub files/repos, arxiv, gists, PDFs, HTML) into clean markdown via contextkit
- Caches everything in SQLite via litesearch to never burn quota twice
- Exposes `search()` and `fetch()` as the entire public API
- CLI via fastcore `call_parse`
- Skill interface (`allow()`) for Hermes / pyskills agent frameworks

**Design:** fastai-style throughout. Two files. ~500 lines total. Read like a notebook.

### 3.1 Core Primitives with Correct Examples

**`store_attr` for every `__init__` — no manual `self.x = x`:**
```python
# WRONG
class QuotaManager:
    def __init__(self, quota_file, providers):
        self.quota_file = quota_file
        self.providers = providers

# RIGHT
class QuotaManager:
    def __init__(self, quota_file='~/.webba/quota.json', providers=None):
        store_attr()
        self.providers = ifnone(providers, PROVIDERS)
        self._load()
```

**`@delegates` — threads params from one function into another's `**kwargs`:**

The THREE rules (from fastcore docs, non-negotiable):
1. The decorated function MUST accept `**kwargs`
2. The decorated function MUST forward `**kwargs` to the delegatee internally
3. Only params WITH defaults are pulled in — required/positional params are never delegated

```python
# WRONG — @delegates used but kwargs never forwarded to QuotaManager
@delegates(QuotaManager)
def search(q:str, n:int=10, **kwargs) -> SearchResults:
    qm = QuotaManager()          # kwargs not forwarded — @delegates is a lie here
    ...

# WRONG — @delegates on _cli with call_parse
# Per fastcore docs: delegates does NOT work with call_parse (sort_args conflict)
@delegates(something)
@call_parse
def _cli(q:str, **kwargs): ...   # NEVER do this

# RIGHT — fetch() delegates read_gh_repo params (branch, as_dict, included_patterns)
@delegates(read_gh_repo, but=['url'])
def fetch(url:str, sel:str=None, heavy:bool=False, cdp:bool=False, **kwargs) -> str|dict:
    "Fetch any URL as clean text. kwargs forwarded to contextkit for gh_repo/gh_file URLs."
    t = _url_type(url)
    if t == 'gh_repo': return read_gh_repo(url, **kwargs)    # forwarded here — valid
    if t == 'gh_file': return read_gh_file(url)
    ...

# RIGHT — _fetch_html delegates contextkit read_link params (heavy, useJina, sel)
@delegates(read_link, but=['url'])
def _fetch_html(url:str, **kwargs) -> str:
    "Fetch HTML. Tier cascade falls through to contextkit read_link with same kwargs."
    try:
        r = niquests.get(url, headers=_random_ua(), timeout=15)
        if r.status_code == 200 and len(r.text) > 500:
            return _extract(r.text, kwargs.get('sel'))
    except Exception: pass
    return read_link(url, **kwargs)   # kwargs forwarded — this makes @delegates valid
```

`but=['url']` excludes `url` from the delegated params (it's already an explicit param).

After applying `@delegates(read_gh_repo, but=['url'])`, the signature of `fetch()` will show:
`(url, sel=None, heavy=False, cdp=False, *, branch=None, as_dict=False, included_patterns=None, ...)` — all of read_gh_repo's defaulted params become visible to callers and IDEs.

**`@patch` for cohesion without inheritance:**
```python
@patch
def to_md(self:SearchResults) -> str:
    "Format results as markdown numbered list."
    return '\n'.join(f'{i+1}. **[{r.title}]({r.url})**  \n   {r.snippet}'
                     for i, r in enumerate(self))

@patch
def fetch(self:Result, sel:str=None, heavy:bool=False) -> str:
    "Fetch full page content for this result."
    from .fetch import fetch as _fetch
    return _fetch(self.url, sel=sel, heavy=heavy)
```

**`L` everywhere — never raw list ops:**
```python
# WRONG
results = []
for r in raw:
    results.append(Result(title=r['title'], url=r['link'], snippet=r['snippet']))

# RIGHT
results = L(raw).map(lambda r: Result(title=r['title'], url=r['link'],
                                       snippet=r.get('snippet',''), provider='serper'))
available = L(PROVIDERS).filter(lambda p: os.environ.get(PROVIDERS[p].env)
                                          or not PROVIDERS[p].needs_key)
```

**`ifnone` — never `or` for defaults:**
```python
limit = ifnone(limit, 10)          # RIGHT — 0 stays 0
limit = limit or 10                # WRONG — falsy 0 becomes 10
```

**`AttrDict` for all config:**
```python
PROVIDERS = {
    'serper': AttrDict(name='serper', base='https://google.serper.dev/search',
                       free_quota=2500, resets='monthly', env='SERPER_API_KEY', needs_key=True),
}
```

**One-liner docstrings only:**
```python
def route(q:str, quota=None) -> str:
    "Pick best available provider for `q` based on intent signals and remaining quota."
```

### 3.2 Anti-Patterns — Hard Block

- No `super().__init__()` chains
- No `self.x = kwargs.get('x', default)` — use `store_attr` + `ifnone`
- No `[x for x in lst]` — use `L(lst).map()`
- No `[x for x in lst if f(x)]` — use `L(lst).filter()`
- No `:param x:` / `:returns:` docblock sections
- No manually maintained `__all__`
- No classes where a function suffices
- No `@delegates` where `**kwargs` are not actually forwarded to the delegatee

### 6.6 `SearchCache`

```python
class SearchCache:
    def __init__(self,
                 db_path:str='~/.webba/cache.db',
                 ttl:int=3600,
    ):
        "SQLite cache via litesearch. hash=True: content-addressed by query string."
        store_attr()
        from litesearch import database
        self.db = database(Path(db_path).expanduser())
        self.store = self.db.get_store('results', hash=True)

    def get(self, key:str) -> list|None:
        "Return cached results if within ttl, else None."

    def set(self, key:str, results:list):
        "Store results under key with current timestamp."
```

**litesearch notes:**
- `database()` is a function, not a class — call as `database(path)`
- `get_store()` is `@patch`ed onto `Database` — call as `db.get_store('results', hash=True)`
- `hash=True` → content-addressed rows; identical query strings hit same row, zero duplicates
- Cache key format: `f"{provider}:{q}:{n}"` — same query, different providers, different rows

### 6.7 SearXNG Setup via dockeasy

**Critical dockeasy facts (verified from core.py source):**
- `drun(img, detach, ports, name, remove, command, check)` — **NO volumes param**
- For volume mounts, the only correct path is `Compose().svc(..., volumes={...}).up()`
- `Compose.svc(name, **kw)` calls `service(**kw)` which accepts `image, build, ports, env, volumes, depends_on, command, **kw`
- `service()` converts dicts: `volumes={'/host': '/container'}` → `['/host:/container']` via `dict2str`
- `service()` converts: `ports={'8080': '8080'}` → `['8080:8080']`
- `service()` converts: `env={'KEY': 'val'}` → `['KEY=val']` via `dict2str(env, '=')`
- Extra `**kw` in `.svc()` pass through to YAML — use for `container_name`
- `Compose.up(detach=True, path=...)` saves YAML then runs `docker compose up -d`
- `docker compose up -d` is **idempotent** — safe no-op if container already running
- `containers(all=False)` returns list of running container name strings

```python
_SEARXNG_URL  = 'http://localhost:8080'

_COMPOSE_PATH = str(_SEARXNG_DIR / 'docker-compose.yml')

def _ensure_searxng() -> str:
    "Start SearXNG via Compose if not running; write settings.yml; return base URL."
    from dockeasy import Compose, containers
    # 1. Honour explicit env override (production deployment)
    url = os.environ.get('SEARXNG_URL')
    if url: return url
    # 2. Write settings — always refresh so config changes are picked up
    _SEARXNG_DIR.mkdir(parents=True, exist_ok=True)
    (_SEARXNG_DIR / 'settings.yml').write_text(_SEARXNG_SETTINGS)
    # 3. Compose up — idempotent, no-op if already running
    (Compose()
        .svc('searxng',
             image='searxng/searxng:latest',
             ports={'8080': '8080'},
             volumes={str(_SEARXNG_DIR): '/etc/searxng'},
             env={'SEARXNG_SECRET': 'webba'},
             container_name=_SEARXNG_NAME)
        .up(detach=True, path=_COMPOSE_PATH)
    )
    # 4. Poll until healthy (~3s cold start)
    _wait_for_searxng(_SEARXNG_URL)
    return _SEARXNG_URL

def _wait_for_searxng(url:str, timeout:int=20, interval:float=0.5):
    "Poll SearXNG / until HTTP 200 or raise RuntimeError on timeout."
    deadline = time.monotonic() + timeout
    while time.monotonic() < deadline:
        try:
            if niquests.get(url, timeout=2).status_code == 200: return
        except Exception: pass
        time.sleep(interval)
    raise RuntimeError(f'SearXNG did not start within {timeout}s at {url}')
```

Container persists across Python sessions. `docker compose up -d` is safe to call every time — after first start it returns in <100ms.

**`_google_scrape` selectolax implementation:**
```python
def _google_scrape(q:str, n:int=10) -> L:
    "Scrape Google results via niquests + selectolax. Fragile CSS selectors — last free resort."
    from selectolax.parser import HTMLParser
    try:
        url = f'https://www.google.com/search?q={niquests.utils.quote(q)}&num={n}&hl=en&gl=us'
        r = niquests.get(url, headers=_random_ua(), timeout=10)
        if r.status_code != 200: return L()
        tree = HTMLParser(r.text)
        results = L()
        for block in tree.css('div.g'):
            a    = block.css_first('a[href]')
            h    = block.css_first('h3')
            snip = block.css_first('.VwiC3b') or block.css_first('[data-sncf]')
            if not (a and h): continue
            href = a.attrs.get('href', '')
            if not href.startswith('http'): continue
            results.append(Result(title=h.text(), url=href,
                                  snippet=snip.text() if snip else '',
                                  provider='google_scrape', ts=time.time()))
        return results[:n]
    except Exception: return L()
```

Google rotates CSS class names ~every 6 months. When `_google_scrape` starts returning `L()`, update `'.VwiC3b'` to current snippet class. Expected maintenance.

`_random_ua()` returns a rotating UA header dict — define at module level for reuse in fetch.py too.

### 6.10 `route()`

```python
def route(q:str, quota:QuotaManager=None) -> str:
    "Pick best available provider for `q` by intent signals, quota, and key availability."
```

Logic:
1. Detect intent: `L(INTENT.items()).filter(lambda kv: any(s in q.lower() for s in kv[1]))`
2. Build preferred list from `INTENT_MAP` for detected intents (deduped, ordered)
3. Append `FREE_TIER_ORDER` as fallback after intent-matched providers
4. `qm = ifnone(quota, QuotaManager())`; filter list by `qm.available()`
5. Skip `fragile=True` providers unless they are the only option left
6. Return first match; absolute fallback is `'ddg'` (always available, never raises)

### 6.12 `search()` — Main Entry Point

**No `@delegates` here** — `search()` is the top-level API. Its signature must be explicit and self-documenting. Provider-specific kwargs are not forwarded to a single delegatee (the provider is chosen dynamically), so `@delegates` would be misleading.

```python
def search(q:str,
           n:int=10,
           provider:str='auto',
           cache:bool=True,
           quota_file:str='~/.webba/quota.json',
           cache_ttl:int=3600,
) -> SearchResults:
    "Search the web. Smart routing, quota tracking, SQLite cache. Zero API keys needed."
```

Flow:
1. `qm = QuotaManager(quota_file=quota_file)`; `sc = SearchCache(ttl=cache_ttl)`
2. Cache key = `f"{provider}:{q}:{n}"` → `sc.get(key)` → return `SearchResults(cached)` if fresh
3. If `provider='auto'`: `p = route(q, quota=qm)` else `p = provider`
4. If `provider='all'`: call all `qm.available()` providers → `rerank()` merged results
5. Otherwise: `results = _PROVIDER_FNS[p](q, n)` → `qm.consume(p)` → `sc.set(key, results)`
6. Return `SearchResults(results)`

### 7.3 URL Type Detection

```python
_GH_FILE = re.compile(r'https://github\.com/[^/]+/[^/]+/blob/')
_GH_REPO = re.compile(r'https://github\.com/[^/]+/[^/]+/?$')
_GIST    = re.compile(r'https://gist\.github\.com/')
_ARXIV   = re.compile(r'(arxiv\.org|ar5iv\.org)')
_DOCS    = re.compile(r'(readthedocs|docs\.|/docs/|\.readthedocs\.io)')

def _url_type(url:str) -> str:
    "Classify URL: gh_file|gh_repo|gist|arxiv|pdf|docs|html|local"
    if not url.startswith('http'): return 'local'
    if _GH_FILE.search(url): return 'gh_file'
    if _GH_REPO.search(url): return 'gh_repo'
    if _GIST.search(url):    return 'gist'
    if _ARXIV.search(url):   return 'arxiv'
    if url.endswith('.pdf'): return 'pdf'
    if _DOCS.search(url):    return 'docs'
    return 'html'
```

### 7.4 HTML Extraction

```python
def _extract(html:str, sel:str=None) -> str:
    "Extract clean text from HTML via selectolax. Removes scripts/nav/footer."
    tree = HTMLParser(html)
    for tag in tree.css('script,style,nav,footer,[role=navigation],header'): tag.decompose()
    node = (tree.css_first(sel) if sel
            else tree.css_first('main,article,[role=main]') or tree.body)
    return node.text(separator='\n', strip=True) if node else ''
```

### 7.5 HTML Fetch Tier Cascade

```
Tier 1: niquests + _random_ua()             — fast, zero deps
Tier 2: Jina reader r.jina.ai/{url}         — bot-bypass, free
Tier 3: contextkit read_link(heavy=True)    — playwrightnb headless (JS-heavy pages)
Tier 4: fastcdp (optional dep)              — attach to running Chrome (auth-gated pages)
```

Tier 3 vs 4 distinction:
- Tier 3 `playwrightnb`: spins fresh headless browser, no login state, handles JS rendering
- Tier 4 `fastcdp`: attaches to YOUR running Chrome via CDP — for pages requiring active login. Requires `pip install webba[cdp]`

`@delegates(read_link, but=['url'])` is correct here — `_fetch_html` falls through to `read_link(url, **kwargs)` in Tier 3, so all of read_link's defaulted params (heavy, useJina, sel) become visible in `_fetch_html`'s signature.

```python
@delegates(read_link, but=['url'])
def _fetch_html(url:str, **kwargs) -> str:
    "Fetch HTML. Tier cascade: niquests → Jina → contextkit read_link (with same kwargs)."
    sel = kwargs.get('sel')
    cdp = kwargs.pop('cdp', False)
    if cdp:
        from fastcdp import fetch as cdp_fetch
        return cdp_fetch(url, sel=sel)
    # Tier 1: niquests
    try:
        r = niquests.get(url, headers=_random_ua(), timeout=15)
        if r.status_code == 200 and len(r.text) > 500:
            return _extract(r.text, sel)
    except Exception: pass
    # Tier 2: Jina
    try:
        r = niquests.get(f'https://r.jina.ai/{url}', timeout=20)
        if r.status_code == 200: return r.text
    except Exception: pass
    # Tier 3: playwrightnb via contextkit — kwargs forwarded here (this validates @delegates)
    try: return read_link(url, **kwargs)
    except Exception: return ''
```

### 7.6 docs.llms.txt Cascade

```python
def _fetch_docs(url:str, sel:str=None) -> str:
    "Try llms-full.txt → llms.txt → plain HTML for documentation URLs."
    base = url.rstrip('/').split('/docs')[0] if '/docs' in url else '/'.join(url.split('/')[:3])
    for suffix in ('/llms-full.txt', '/llms.txt'):
        try:
            r = niquests.get(base + suffix, timeout=10)
            if r.status_code == 200: return r.text
        except Exception: pass
    return _fetch_html(url, sel=sel)
```

### 7.7 `fetch()` — Main Entry Point

`@delegates(read_gh_repo, but=['url'])` is correct here — when `t == 'gh_repo'`, kwargs are forwarded directly to `read_gh_repo(url, **kwargs)`, so its defaulted params (branch, as_dict, included_patterns, excluded_patterns) become visible in `fetch()`'s signature.

```python
@delegates(read_gh_repo, but=['url'])
def fetch(url:str,
          sel:str=None,
          heavy:bool=False,
          cdp:bool=False,
          save_pdf:bool=False,
          **kwargs,
) -> str|dict:
    "Fetch any URL or local path as clean text. kwargs forwarded to read_gh_repo for repo URLs."
    t = _url_type(url)
    if t == 'gh_file':  return read_gh_file(url)
    if t == 'gh_repo':  return read_gh_repo(url, **kwargs)       # delegation valid here
    if t == 'gist':     return read_gist(url)
    if t == 'arxiv':    return read_arxiv(url, save_pdf=save_pdf)
    if t == 'pdf':
        import tempfile
        with tempfile.NamedTemporaryFile(suffix='.pdf', delete=False) as f:
            f.write(niquests.get(url, timeout=30).content); tmp = f.name
        from contextkit.read import read_pdf
        return read_pdf(tmp)
    if t == 'docs':     return _fetch_docs(url, sel=sel)
    if t == 'local':    return Path(url).read_text()
    return _fetch_html(url, sel=sel, heavy=heavy, cdp=cdp)
```

After `@delegates(read_gh_repo, but=['url'])`, the signature of `fetch()` shows:
`(url, sel=None, heavy=False, cdp=False, save_pdf=False, *, branch=None, as_dict=False, included_patterns=None, ...)` — repo params are discoverable in IDE tab completion.

## 16. delegates Correctness Summary

From fastcore docs (https://fastcore.fast.ai/meta.html):

| Rule | Detail |
|---|---|
| Decorated fn must have `**kwargs` | Non-negotiable. No `**kwargs` = `@delegates` does nothing |
| `**kwargs` must be forwarded to delegatee | If you don't call `delegatee(..., **kwargs)` internally, `@delegates` is lying |
| Only defaulted params are delegated | Required/positional params are never pulled in |
| Works with functions and class methods | Including `@classmethod`, instance methods, and between classes |
| Does NOT work with `call_parse` | Per fastcore docs. Use explicit `Param` annotations for CLI functions |
| `but=['param']` excludes specific params | Use to exclude params you've already declared explicitly |

In webba, `@delegates` is used in exactly two places:
- `_fetch_html` delegates to `read_link` — valid because Tier 3 forwards `**kwargs` to `read_link(url, **kwargs)`
- `fetch` delegates to `read_gh_repo` — valid because gh_repo branch forwards `**kwargs` to `read_gh_repo(url, **kwargs)`

Everywhere else, explicit signatures are used. `search()` has no `@delegates` because the called provider is chosen dynamically at runtime.
